# Gustavo Santos — Portfolio

Portfólio pessoal com direção visual “Titanium”: uma experiência editorial escura, responsiva e rica em movimento.

## Destaques

- animações de entrada e parallax com respeito a `prefers-reduced-motion`;
- cursor magnético, cards com tilt e partículas em canvas;
- projetos, certificados e currículo em uma navegação acessível;
- layout responsivo para desktop, tablet e celular;
- card social exclusivo para compartilhamento.

Para gerar a versão de produção, use `npm run build`.
